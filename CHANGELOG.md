## 0.1.0

- Language registration for `.cadabra` files
- Syntax highlighting (HCL-based with cadabra-specific type keywords)
- Custom file icons (light and dark themes)
- Code snippets for all block types (db, table, column, index, fk, enum, check)
- Bracket matching and auto-closing
- Comment toggling (# and /* */)
- Code folding
- Auto-indentation for block types
